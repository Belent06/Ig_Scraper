# Instagram Scraper 📸
Script en Python desarrollado con la librería **Instaloader** para la extracción de metadatos y publicaciones.

### Tecnologías:
* Python 3.x
* Instaloader
* Manejo de sesiones mediante cookies (Netscape format)

### Nota de seguridad:
Este repositorio no incluye archivos de sesión ni cookies por motivos de privacidad.
